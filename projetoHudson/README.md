# Projeto

### Instruções de compilação
- Para inserir os usuários em lote, vocẽ deve modificar na main.c o caminho do arquivo que quer inserir(1000, 10000, ou 10000)
- Para compilar, vá até a pasta src e abra a mesma no terminal
- Digite: 
''' 
    gcc main.c bloom_filter.c hash.c -o main
'''

### Formato de Entrada
- Não possui entradas, apenas configurações antes da compilação.

### Exemplos de execução
- Vai aparecer um menu iniciar, pode ficar a vontadade para usar.
- No menu você vai ter as opções de estatísticas
